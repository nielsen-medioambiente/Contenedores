⚙️ Configurar URL de Web App
Pegá la URL de tu implementación como "Aplicación web" de Apps Script.
Cómo obtener la URL:
1. Abrí el archivo Google Sheets
2. Extensiones → Apps Script
3. Pegá el código AppScript_ResTrack.gs
4. Implementar → Nueva implementación
5. Tipo: Aplicación web
6. Ejecutar como: Yo (tu cuenta)
7. Acceso: Cualquier usuario (o solo vos)
8. Copiá la URL y pegala arriba
